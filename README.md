## How to run
1. Download all files and put them in the same folder.
2. Open CMD and type [python [**file path**]\main_gmpe_south_korea.py]

## Library required
1. PySimpleGUI == 4.49.0
2. pickle == 4.0
3. numpy == 1.20.2
4. pandas == 1.2.0
5. matplotlib == 3.3.2
6. sklearn == 0.23.2
